# Model Card: ai-pricing-service model_v1

## Назначение

Модель `model_v1` прогнозирует спрос товара в единицах (`demand_units`) для демонстрационного pricing-сервиса.

## Тип модели

```text
RandomForestRegressor
```

Модель сохранена как sklearn pipeline:

```text
DictVectorizer -> RandomForestRegressor
```

## Runtime-артефакты

- `artifacts/model_v1.pkl` - предобученная модель;
- `artifacts/scaler_v1.pkl` - вспомогательная статистика цен;
- `artifacts/model_metadata.json` - версия, признаки, метрики и checksum.

## Входные признаки

- `product_id` - идентификатор товара;
- `horizon_days` - горизонт прогноза, от 1 до 14 дней;
- `current_base_price` - текущая базовая цена;
- `is_weekend` - признак выходного дня.

## Выход модели

```text
predicted_demand_units
```

`recommended_price` не является прямым выходом модели. Это результат бизнес-логики, которая использует прогноз спроса.

## Данные

Модель обучена на синтетическом датасете:

```text
synthetic-pricing-v1-seed-42
```

Синтетические данные подходят для демонстрации архитектуры, но не заменяют реальные исторические продажи.

## Метрики

Текущие метрики на holdout-выборке:

```text
MAE:  2.324
RMSE: 2.957
```

Актуальные значения хранятся в `artifacts/model_metadata.json`.

## Контракт модели

Сервис проверяет при загрузке:

- версию модели;
- тип модели;
- список признаков;
- target;
- checksum модели;
- checksum scaler;
- успешное test prediction.

Контракт доступен через API:

```text
GET /v1/model/info
```

Компактная версия сервиса и модели:

```text
GET /version
```

Эвристическое объяснение конкретного прогноза:

```text
POST /v1/explain
```

Batch inference:

```text
POST /v1/batch-predict
```

История последних запросов:

```text
GET /v1/predictions/recent
```

История хранится в памяти процесса и предназначена только для демонстрации.

## Ограничения

- модель обучена на синтетических данных;
- нет реальной истории продаж;
- нет сезонности, промо-акций, остатков на складе и цен конкурентов;
- unknown `product_id` обрабатывается технически корректно, но качество такого прогноза ограничено;
- confidence score является инженерной оценкой, а не статистически откалиброванной вероятностью.
